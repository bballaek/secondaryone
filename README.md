# Secondary 1 File System
ระบบจัดการไฟล์เรียนสำหรับนักเรียน ม.1 โรงเรียนเซนต์คาเบรียล

## ฟีเจอร์
- ดูและดาวน์โหลดไฟล์จาก Google Drive
- ค้นหาไฟล์
- กรองตามวิชา
- แสดงสถิติไฟล์

## การใช้งาน
1. เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์
2. ไฟล์จะโหลดจาก Google Drive อัตโนมัติ
